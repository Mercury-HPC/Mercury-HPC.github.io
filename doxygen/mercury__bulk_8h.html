HG_EXPORT hg_return_t HG_Bulk_create ( hg_class_t *  hg_class,

void **  buf_ptrs,
const hg_size_t *  buf_sizes,
hg_uint8_t  flags,
hg_bulk_t *  handle 
)

Create an abstract bulk handle from specified memory segments.

Memory allocated is then freed when HG_Bulk_free() is called.

Remarks
If NULL is passed to buf_ptrs, i.e., 
HG_Bulk_create(count, NULL, buf_sizes, flags, &handle)
memory for the missing buf_ptrs array will be internally allocated.
Parameters
hg_class[IN] pointer to HG class
count[IN] number of segments
buf_ptrs[IN] array of pointers
buf_sizes[IN] array of sizes
flags[IN] permission flag:
  • HG_BULK_READWRITE
  • HG_BULK_READ_ONLY
  • HG_BULK_WRITE_ONLY
handle[OUT] pointer to returned abstract bulk handle
Returns
HG_SUCCESS or corresponding HG error code

)

Access bulk handle to retrieve memory segments abstracted by handle.

Remarks
When using mercury in co-resident mode (i.e., when addr passed is self addr), this function allows to avoid copy of bulk data by directly accessing pointers from an existing HG bulk handle.
Parameters
handle[IN] abstract bulk handle
offset[IN] bulk offset
size[IN] bulk size
flags[IN] permission flag:
  • HG_BULK_READWRITE
  • HG_BULK_READ_ONLY
max_count[IN] maximum number of segments to be returned
buf_ptrs[IN/OUT] array of buffer pointers
buf_sizes[IN/OUT] array of buffer sizes
actual_count[OUT] actual number of segments returned
Returns
HG_SUCCESS or corresponding HG error code
