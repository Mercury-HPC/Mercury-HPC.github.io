HG_EXPORT hg_return_t HG_Forward ( hg_handle_t  handle,
hg_cb_t  callback,
void *  arg,
void *  in_struct 
)

Forward a call to a local/remote target using an existing HG handle.

Input structure can be passed and parameters serialized using a previously registered input proc. After completion, user callback is placed into a completion queue and can be triggered using HG_Trigger(). RPC output can be queried using HG_Get_output() and freed using HG_Free_output().

Remarks
This routine is internally equivalent to:
  • HG_Core_get_input()
  • Call hg_proc to serialize parameters
  • HG_Core_forward()
Parameters
handle[IN] HG handle
callback[IN] pointer to function callback
arg[IN] pointer to data passed to callback
in_struct[IN] pointer to input structure

Try to progress RPC execution for at most timeout until timeout is reached or any completion has occurred.
